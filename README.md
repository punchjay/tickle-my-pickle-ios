# Tickle My Pickle (iOS)

Native SwiftUI pickleball court finder. Geocode a free-text location (or device
geolocation) -> Google Places (New) text search around it -> numbered pins on
an Apple MapKit map + a results list with rating, open/closed status, inferred
amenity badges, directions, and on-device favorites.

Successor to the Expo/React Native `tickle-my-pickle-mobile` app -- this is a
ground-up native rewrite, not a port of that codebase. It shares no code with
it. See the sibling `tickle-my-pickle` repo for the original Vite web app this
product concept started from.

## Why native, why now

The RN app hit real friction: Google Maps API key management split three ways
(web-referrer-restricted key, a separate REST key for server-side Geocoding
/Places calls, native app restrictions), plus a growing `.native.tsx`/`.web.tsx`
file surface for a fundamentally small app. This app avoids that by using
Apple MapKit (no key, no dependency) for the map itself, and a single
iOS-app-restricted Google key for search data only.

## Stack

SwiftUI, iOS 17+, Swift 6 (strict concurrency). No Expo, no React Native, no
Google Maps SDK. Project file is generated by [XcodeGen](https://github.com/yonaskolb/XcodeGen)
from `project.yml` -- the Xcode project itself is not checked in, so the repo
stays plain-text/VSCode-friendly.

## Setup

```bash
brew install xcodegen
cp Secrets.xcconfig.example Secrets.xcconfig   # then fill in a real key -- see comments in that file
xcodegen generate
```

## Build & run (Simulator, no Xcode GUI needed)

```bash
xcodebuild -project TickleMyPickle.xcodeproj -scheme TickleMyPickle -configuration Debug \
  -destination 'platform=iOS Simulator,name=iPhone 17,OS=latest' -derivedDataPath build build

xcrun simctl boot "iPhone 17" 2>/dev/null || true
xcrun simctl bootstatus "iPhone 17" -b
xcrun simctl install booted build/Build/Products/Debug-iphonesimulator/TickleMyPickle.app
xcrun simctl launch booted com.punchjay.ticklemypickle
```

## Tests

```bash
xcodebuild test -project TickleMyPickle.xcodeproj -scheme TickleMyPickle \
  -destination 'platform=iOS Simulator,name=iPhone 17,OS=latest' -derivedDataPath build
```
